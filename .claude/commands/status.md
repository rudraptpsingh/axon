Quick project status check.

Run these and report:
1. `cargo test --workspace` — pass/fail count
2. `axon query hw_snapshot` — current CPU, RAM, headroom level
3. `axon query session_health` — recent alerts and worst impact
4. `git log --oneline -5` — last 5 commits
5. `git status` — current branch, staged/unstaged changes

Summarize: build health, machine health, repo state.
